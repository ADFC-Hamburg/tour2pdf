# ADFC Tour2pdf

Ein Script aus dem ADFC Tourenportal ein PDF macht.

# Development


Instalation: Linux mit Python
```
./create-venv.sh

```

Starten mit:
```
./.venv/bin/fastapi dev ./tour2pdf.py
```

Dann einen Browser auf:

http://127.0.0.1:8000/

aufrufen.
